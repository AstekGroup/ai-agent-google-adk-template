# Custom Agent Template

Agent personnalisé avec BaseAgent pour logique d'orchestration avancée.

## Architecture

```
User → Runner → CustomAgent → Custom Logic → Response
```

## Installation

```bash
poetry install
```

## Utilisation

Ce template démontre comment créer un agent avec logique d'orchestration personnalisée utilisant BaseAgent.

## Cas d'usage

- Workflows complexes personnalisés
- Logique conditionnelle avancée
- Intégrations spécifiques nécessitant contrôle total
